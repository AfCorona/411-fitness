# 411-fitness

Swift fitness and nutrition application

#installation


1. Install Xocde on macOS
2. go to https://github.com/AfCorona/411-fitness and download zip
3. extract and move to preferred location or leave as is
4. Open xcode and click open existing project
5. On the top of the screen go to product and hit run
6. wait for everything to install and run 
7. Finished.
